# Phantom Sans Font Files

Place Phantom Sans font files here:

- `PhantomSans-Regular.woff2`
- `PhantomSans-Regular.woff`
- `PhantomSans-Bold.woff2`
- `PhantomSans-Bold.woff`

These are Hack Club's brand typeface. If the files are not available,
the system-ui fallback stack will be used automatically (which matches
what Hack Club's theme specifies as fallback).

The @font-face declarations in `src/app/globals.css` reference these paths.
